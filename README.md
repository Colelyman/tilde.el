# General:
```
SPC /     : ripgrep
SPC TAB   : previous buffer
SPC SPC   : Open M-x
```
# Files:
```
SPC p f   : find files
```
# Buffers:
```
SPC b b   : buffers list
```
# Window:
```
SPC w l   : move right
SPC w h   : move left
SPC w j   : move down
SPC w k   : move up
SPC w /   : split right
SPC w -   : split bottom
SPC w x   : close window
```
# Other:
```
SPC a t   : open terminal in current buffer
```
